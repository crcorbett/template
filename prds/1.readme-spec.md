### 8. Known Issues

#### Nitro Disabled in Dev Server

Document the HTTP/2 + Transfer-Encoding bug:

- **Issue**: When running with Bun + Nitro + Vite in development, HTTP/2 causes Transfer-Encoding errors
- **Workaround**: Nitro is only enabled during `build` command, disabled during `dev`
- **Impact**: None for deployments - Vercel builds work correctly
- **Reference**: https://github.com/TanStack/router/issues/6050
- **Code example** from `apps/admin/vite.config.ts`:

```typescript
command === "build"
  ? nitro({
      preset: "vercel",
      vercel: {
        functions: {
          runtime: "bun1.x",
        },
      },
    })
  : null,
```
